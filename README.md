# Adventure of Dust

- 설명 : 2D 점프 게임
- 개요 : 캐릭터를 움직여 장애물을 피하고 부수며 스테이지를 클리어
- 게임 방법: 터치패드로 좌우 움직임과 점프 키로 플레이
- 개발 환경 : Unity 2022.3.2f1
- 개발 기간 2023.10.23 ~ 2023.12.14
